# vite-fait

```js
// (num, 5), (num, 4.5) -> (str, 'true')
Boolean(5 - 4.5).toString();
  // (num, 5), (num, 4.5)
  5 - 4.5;
  // (num, .5)
  Boolean(.5);
  // (bool, true)
  true.toString();
  // (str, 'true')

```
